# SkillSwap

**SkillSwap** is a simple web app where users can exchange skills with each other. Users can register, offer skills, request swaps, give feedback, and admins can manage users.

## Features

- User registration and login (mocked using user ID)
- Create and view public profiles
- Search users by skill
- Request skill swaps
- Accept, reject, or delete swap requests
- Submit feedback to other users
- Admin panel:
  - Ban users
  - Clear inappropriate skills
  - View reports of users and swaps

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express, MongoDB, Mongoose
